# lalamove-nodejs-sdk

## GetStarted

```bash
npm i --save lalamove-nodejs-sdk
```

```js
const Lalamove = require('lalamove-nodejs-sdk');

const lalamove = new Lalamove({
  apiSecret: YOUR_API_SECRET,
  apiKey: YOUR_API_KEY,
  market: YOUR_MARKET,
});

```
